Задание 2. Задание на урок

Используя макет страницы (см. примеры на сайте Полякова), создать файл maket.html и style_m.css. Информацию и картинки можно найти в Интернете или в архиве на том же сайте Полякова. Переход на файл вставить в index.html.

Задание 3. Список заданий по информатике

Подготовить страницу с разбором заданий по одной из тем информатики. Включить:

  • Общая информация
  • Теория для решения
  • Разбор 3-5 заданий
  • Самопроверка 5-10 заданий (используя элементы управления form и javascript)
  • Список источников
